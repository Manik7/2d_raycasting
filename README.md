# Box raycasting

Render an image of a box and a ground plane using raycasting.

## Getting started

1. Install dependencies
```bash
sudo apt get install libgtest-dev cmake g++
```

2. Clone the repo
```bash
git clone #...
```

3. Build and run:
```bash
# out-of-source build
cd box_raycasting && mkdir out && cd out && cmake .. && cd ..
cmake --build out/ && ./out/unit_tests
```
